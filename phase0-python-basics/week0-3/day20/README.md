# Day 20: Phase 0 종합 — list·dict·함수·파일이 들어간 스크립트

## 🎯 오늘의 목표

- **리스트, 딕셔너리, 함수, 파일**이 모두 들어간 **작은 스크립트 하나**를 직접 만든다.

---

## ✅ 오늘 할 일 (순서대로)

### 1. 주제 정하기

- **할 일**: 아래 중 **하나**를 골라서 (또는 비슷한 아이디어로) 스크립트를 만든다.
  - **옵션 A**: 텍스트 파일을 한 줄씩 읽어서, 단어별로 갯수를 세고 딕셔너리로 만든 뒤, 가장 많이 나온 단어 1개를 출력한다. (list, dict, 함수, 파일 사용)
  - **옵션 B**: 로그 파일을 읽어서 "[ERROR]"가 포함된 줄만 골라 리스트에 모은 뒤, 개수를 출력한다. (list, 함수, 파일, 문자열)
  - **옵션 C**: 숫자 리스트를 인자로 받는 함수가 합계·평균을 구해 딕셔너리로 return하고, 그걸 print 한다. (list, dict, 함수)
- **완료 조건**: 위 네 가지(리스트, 딕셔너리, 함수, 파일) 중 **최소 3가지**가 한 스크립트에 들어 있다.

### 2. 실행·동작 확인

- **할 일**: `day20/` 안에 `project.py`(이름 자유)로 저장하고, `uv run python project.py` 로 실행해 에러 없이 결과가 나오는지 확인한다.
- **완료 조건**: 스크립트가 실행되고, 기대한 결과(예: 단어 개수, ERROR 줄 개수, 합계/평균)가 출력된다.
