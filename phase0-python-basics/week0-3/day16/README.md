# Day 16: 파일 쓰기 — write(), with open() as f, 한 줄씩 처리

## 🎯 오늘의 목표

- **파일에 쓰기** (write)를 해 보고, **with open()**으로 열어 **한 줄씩 읽어서 처리**하는 흐름을 써 본다.

---

## ✅ 오늘 할 일 (순서대로)

### 1. write()로 파일 쓰기

- **할 일**: `day16/` 안에 `write_file.py`를 만든다.
  - `with open("output.txt", "w") as f:` 로 **쓰기 모드**로 연다. 안에서 `f.write("Hello\n")`, `f.write("World\n")` 를 넣어 두 줄이 쓰이게 한다. 실행 후 `output.txt` 파일이 생기고, 열어보면 "Hello", "World" 두 줄이 있는지 확인한다.
- **완료 조건**: "w" 모드로 open하고 write()로 output.txt에 2줄이 써진다.

### 2. 한 줄씩 읽어서 처리

- **할 일**: Day 15의 sample.txt(또는 새 텍스트 파일)를 연다. `for line in f:` 로 한 줄씩 읽으면서, `line.strip()` 한 결과를 리스트에 append 한다. 반복이 끝나면 그 리스트를 print 해 본다. (또는 각 줄을 대문자로 바꿔 출력해 본다.)
- **완료 조건**: 파일을 한 줄씩 읽어 리스트에 모으거나, 각 줄을 조금 바꿔 출력했다.

---

## 📌 참고

- **open(경로, "w")**: 쓰기. **"r"**은 읽기(기본). **write(문자열)** 에는 \n을 직접 넣어야 줄바꿈이 된다.
- **with open(...) as f:** 로 열면 블록 끝에서 자동으로 닫힌다.
