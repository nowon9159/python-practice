# Day 3: 리스트(list) — [], 인덱스, append, len, for

## 🎯 오늘의 목표

- **리스트**를 만들고, **인덱스**로 접근하며, **append**와 **len**, **for**로 하나씩 돌아본다.

---

## ✅ 오늘 할 일 (순서대로)

### 1. 리스트 만들고 인덱스로 접근

- **할 일**: `day03/` 안에 `list_basics.py`를 만든다.
  - `fruits = ["사과", "바나나", "오렌지"]` 로 리스트를 만든다.
  - `print(fruits[0])` → "사과", `print(fruits[1])` → "바나나"가 나오는지 확인한다. (인덱스는 0부터 시작)
  - `len(fruits)` 를 print해서 3이 나오는지 확인한다.
- **완료 조건**: fruits[0], fruits[1], len(fruits)를 실행해 위와 같이 나온다.

### 2. append와 for 루프

- **할 일**: 같은 파일에 아래를 넣는다.
  - `fruits.append("포도")` 후 `print(fruits)` 로 "포도"가 맨 뒤에 붙었는지 확인.
  - `for x in fruits:` 와 그 다음 줄에 들여쓰기해서 `print(x)` 를 넣어, 리스트의 값이 한 줄씩 출력되는지 확인한다.
- **완료 조건**: append 후 리스트가 4개 항목이 되고, for로 네 개가 한 줄씩 출력된다.

---

## 📌 참고

- **리스트**: `[]` 로 만들고, 순서가 있다. **인덱스**: 0이 첫 번째.
- **append(값)**: 리스트 맨 뒤에 추가. **len(리스트)**: 항목 개수.
- **for x in 리스트:** → x에 항목이 하나씩 들어가며 반복.
