# Day 4: 리스트 더 쓰기 — 슬라이싱, in, 리스트 안 리스트

## 🎯 오늘의 목표

- **슬라이싱** `[시작:끝]`, **in**, 그리고 **리스트 안에 리스트**를 한 번씩 써 본다.

---

## ✅ 오늘 할 일 (순서대로)

### 1. 슬라이싱 [시작:끝]

- **할 일**: `day04/` 안에 `list_slice.py`를 만든다.
  - `nums = [10, 20, 30, 40, 50]` 을 만든다.
  - `print(nums[1:4])` → [20, 30, 40] 이 나오는지 확인한다 (1번 인덱스부터 4번 **직전**까지).
  - `print(nums[:3])` (처음부터 3번 직전까지), `print(nums[2:])` (2번부터 끝까지) 도 실행해 본다.
- **완료 조건**: [1:4], [:3], [2:] 결과를 보고 "시작 포함, 끝 미포함"을 이해했다.

### 2. in과 리스트 안 리스트

- **할 일**: 같은 파일에 `if 30 in nums: print("30 있음")` 을 넣어 30이 있을 때만 출력되는지 확인한다.
  - `matrix = [[1, 2], [3, 4]]` 를 만든다. `print(matrix[0])` → [1, 2], `print(matrix[0][1])` → 2 가 나오는지 확인한다.
- **완료 조건**: `in`으로 값 존재 여부를 확인했고, matrix[0][1]로 안쪽 값에 접근했다.

---

## 📌 참고

- **슬라이싱**: [시작:끝] → 시작 인덱스는 포함, 끝 인덱스는 **미포함**.
- **값 in 리스트**: 값이 리스트에 있으면 True.
