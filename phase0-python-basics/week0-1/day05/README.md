# Day 5: 조건문 — if / elif / else, 비교·논리 연산자

## 🎯 오늘의 목표

- **if, elif, else**로 분기하고, **비교 연산자**(==, <, >)와 **논리 연산자**(and, or)를 써 본다.

---

## ✅ 오늘 할 일 (순서대로)

### 1. if / else 작성

- **할 일**: `day05/` 안에 `conditions.py`를 만든다.
  - 변수 `score = 80` 을 둔다.
  - `if score >= 60:` 다음 줄에 들여쓰기해서 `print("합격")` 을 넣는다. `else:` 다음 줄에 들여쓰기해서 `print("불합격")` 을 넣는다. 실행 시 "합격"이 나오는지 확인한다.
  - score를 50으로 바꾼 뒤 다시 실행하면 "불합격"이 나오는지 확인한다.
- **완료 조건**: score에 따라 "합격" 또는 "불합격"이 출력된다.

### 2. elif와 and, or

- **할 일**: 같은 파일에 아래를 추가한다.
  - `if score >= 90: print("A")`  
  - `elif score >= 80: print("B")`  
  - `elif score >= 70: print("C")`  
  - `else: print("D")`  
  score를 85로 두면 "B"가 나오는지 확인한다.
  - `age = 20` 이고 `has_ticket = True` 일 때, `if age >= 18 and has_ticket: print("입장 가능")` 이 한 번 실행되면 "입장 가능"이 나오는지 확인한다.
- **완료 조건**: elif로 A/B/C/D 중 하나가 나오고, and로 "입장 가능"이 나온다.

---

## 📌 참고

- **==** 같다, **!=** 다르다, **<, >, <=, >=** 크기 비교.
- **and**: 둘 다 참일 때만 참. **or**: 하나라도 참이면 참.
