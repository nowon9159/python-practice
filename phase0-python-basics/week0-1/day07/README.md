## ✅ 오늘 할 일 (순서대로)

### 1. 연습 스크립트 하나 작성

- **할 일**: `day07/` 안에 `review.py`를 만든다. 아래를 **한 파일**에 모두 넣어 실행해 본다.
  - 변수: 이름 하나, 숫자 하나 (str, int).
  - 리스트 하나 만들고, append 한 번, for로 한 번 출력.
  - if로 숫자가 10 이상이면 "10 이상", 아니면 "10 미만" 출력.
  - while로 1부터 3까지 출력 (count 변수 사용).
- **완료 조건**: 위 네 가지가 한 파일에 들어 있고, `uv run python review.py` 실행 시 에러 없이 동작한다.

### 2. 막힌 부분 정리

- **할 일**: 헷갈렸던 것(예: 인덱스 0부터, 슬라이싱 끝 미포함)을 `day07/notes.md`에 1~2문장으로 적어 둔다.
- **완료 조건**: 복기할 수 있는 문장이 한 줄이라도 있다.
