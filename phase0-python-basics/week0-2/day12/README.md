# Day 12: 함수 더 쓰기 — 기본값 인자, 리스트/딕셔너리 넘기기

## 🎯 오늘의 목표

- **기본값 인자**와 **여러 return**을 쓰고, **리스트·딕셔너리**를 인자로 넘겨 본다.

---

## ✅ 오늘 할 일 (순서대로)

### 1. 기본값 인자

- **할 일**: `day12/` 안에 `functions2.py`를 만든다.
  - `def say(msg, count=1):` 로 두고, 안에서 `for i in range(count): print(msg)` 를 넣는다. `say("Hi")` → 한 번, `say("Bye", 3)` → 세 번 출력되는지 확인한다.
- **완료 조건**: 기본값으로 한 번, 인자를 바꿔 세 번 출력했다.

### 2. 리스트/딕셔너리 인자

- **할 일**: 같은 파일에 `def sum_list(arr):` 를 만들고, for로 arr의 숫자를 더해 return한다. `print(sum_list([1, 2, 3]))` → 6 이 나오는지 확인한다.
  - (선택) `def get_name(d): return d.get("name", "이름없음")` 처럼 딕셔너리를 받아 키로 값을 반환하는 함수를 만들고 호출해 본다.
- **완료 조건**: 리스트를 인자로 넘겨 합을 반환하는 함수를 실행했다.

---

## 📌 참고

- **def f(a, b=0):** → b를 안 주면 0으로 간주.
- 함수에 리스트·딕셔너리를 넘기면 참조가 넘어가서, 함수 안에서 수정하면 밖에서도 바뀐다.
