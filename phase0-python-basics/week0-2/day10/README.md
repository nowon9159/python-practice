# Day 10: 튜플(tuple), 집합(set) 맛보기

## 🎯 오늘의 목표

- **튜플**은 순서 있지만 수정 불가, **집합**은 순서 없고 중복 없음—차이를 한 번씩 써 본다.

---

## ✅ 오늘 할 일 (순서대로)

### 1. 튜플 만들고 사용

- **할 일**: `day10/` 안에 `tuple_set.py`를 만든다.
  - `t = (1, 2, 3)` 으로 튜플을 만든다. `print(t[0])` → 1, `print(t[1])` → 2 가 나오는지 확인한다.
  - (선택) `t[0] = 10` 처럼 값을 바꾸려고 하면 **에러**가 나는지 확인한다 (튜플은 수정 불가).
- **완료 조건**: 튜플을 만들고 인덱스로 읽었다.

### 2. 집합으로 중복 제거

- **할 일**: 같은 파일에 `nums = [1, 2, 2, 3, 3, 3]` 리스트를 만든다. `s = set(nums)` 로 집합으로 바꾼 뒤 `print(s)` 하면 {1, 2, 3}처럼 중복이 사라졌는지 확인한다.
  - `s.add(4)`, `s.add(2)` 후 `print(s)` 로 2는 한 번만 있고 4가 추가됐는지 확인한다.
- **완료 조건**: set()으로 리스트에서 중복을 제거했고, add()를 한 번 썼다.

---

## 📌 참고

- **튜플**: ( ). 순서 있음, **수정 불가(immutable)**.
- **집합**: set( ). 순서 없음, **중복 없음**. add(), in 사용 가능.
