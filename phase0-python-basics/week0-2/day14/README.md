# Day 14: 주간 복습 — dict·함수·문자열 조합

## 🎯 오늘의 목표

- **딕셔너리, 함수, 문자열**을 섞어서 간단한 **파싱/정리** 예제를 하나 만든다.

---

## ✅ 오늘 할 일 (순서대로)

### 1. 조합 연습 스크립트

- **할 일**: `day14/` 안에 `review_week2.py`를 만든다. 아래 중 **최소 2가지**를 한 파일에 넣어 실행해 본다.
  - **문자열 파싱**: `"name=홍길동,age=20"` 같은 문자열을 split으로 나눈 뒤, "key=value"를 다시 split해 딕셔너리 `{"name": "홍길동", "age": "20"}` 으로 만든다.
  - **함수 + 딕셔너리**: 딕셔너리 하나를 인자로 받아, 키 개수를 return하는 함수를 만들고 호출한다.
  - **함수 + 문자열**: 문자열 하나를 받아 단어 리스트로 split하고, 단어 개수를 return하는 함수를 만들고 호출한다.
- **완료 조건**: 위 중 2가지 이상이 한 파일에 들어 있고, 실행 시 결과가 출력된다.

### 2. 막힌 부분 메모

- **할 일**: 헷갈렸던 점을 `day14/notes.md`에 1~2문장으로 적어 둔다.
- **완료 조건**: 복기할 수 있는 문장이 한 줄이라도 있다.
