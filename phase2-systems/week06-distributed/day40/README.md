- **할 일**: `day40/` 안에 `raft_simple.py`를 만든다. **옵션 A**: 여러 노드(클래스 인스턴스)가 **term**과 **투표**를 주고받는 **리더 선출** 시뮬레이션(타이머/이벤트는 단순화 가능). **옵션 B**: 리더 1개가 있고, 로그 항목을 **팔로워 목록**에 복제하는 **로그 복제**만 구현(동기/비동기 무관). 노드는 2~3개면 충분.
