# Day 38: CAP 정리 · BASE vs ACID 이론

## 🎯 오늘의 목표

## ✅ 오늘 할 일 (순서대로)

### 1. CAP 세 가지 정리

- **할 일**: `day38/` 안에 `cap_base_notes.md`를 만든다. **C, A, P** 각각이 무엇인지 한 문장씩 적는다. "파티션 발생 시 C와 A 중 2개만 만족 가능"한 이유를 2~3문장으로 적는다.
- **완료 조건**: CAP 세 가지 의미를 참고 없이 말할 수 있다.

### 2. ACID vs BASE 비교표

- **할 일**: 같은 파일에 **ACID**(트랜잭션: 원자성, 일관성, 격리, 지속성)와 **BASE**(Basically Available, Soft state, Eventual consistency)를 **표**로 비교한다. "언제 ACID, 언제 BASE" 한 줄씩 추가해도 됨.
- **완료 조건**: 비교표가 문서에 있고, 한 문장이라도 "분산·일관성 완화 시 BASE"가 드러난다.

### 3. (선택) 일관성 vs 가용성 시나리오

- **할 일**: "파티션 시 일관성 선택하면 어떤 현상, 가용성 선택하면 어떤 현상"을 시나리오 1개(2~3문장)로 적는다.
- **완료 조건**: 선택. 적었다면 문서에 포함된다.

### 4. Anki 카드 추가

- **할 일**: "CAP: C, A, P 의미 / 파티션 시 2개만" "ACID vs BASE" **카드 1장** 이상 추가.
