### 2. HTTP 로드밸런서 구현

- **할 일**: `day37/` 안에 `http_load_balancer.py`를 만든다.
  - **동작**: **한 포트**(예: 8080)에서 TCP를 **accept**한다. 받은 데이터를 **HTTP 요청**으로 파싱(또는 그대로 전달)한 뒤, **백엔드 목록**(예: `["127.0.0.1:9001", "127.0.0.1:9002"]`) 중 하나로 **라운드로빈**으로 전달한다. 백엔드에 **connect → send(받은 요청) → recv(응답)** 한 뒤, 그 응답을 클라이언트에게 **send**한다.
  - **조건**: 백엔드가 2개 이상이면 순서대로 번갈아 보내면 됨. (선택) 백엔드 장애 시 다음 백엔드로 시도.
  - **테스트**: 간단한 HTTP 서버 2개를 다른 포트에서 띄우고, 로드밸런서를 통해 요청이 번갈아 가는지 확인.
- **완료 조건**: 한 포트로 들어온 요청이 여러 백엔드로 라운드로빈 분배되고, 클라이언트가 응답을 받는다.
