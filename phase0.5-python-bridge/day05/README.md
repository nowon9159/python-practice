## ✅ 오늘 할 일 (순서대로)

### 1. 타입 힌트 붙인 함수 만들기

- **할 일**: `day05/` 안에 **`typing_basics.py`** 를 만든다.
  - **타입 힌트**는 "이 인자는 이런 타입이다, 반환값은 이런 타입이다"를 적어 두는 것이다.
  - 예: **`def add(a: int, b: int) -> int:`** 그러면 `add` 는 int 두 개를 받아서 int를 반환한다고 표시한 셈이다.
  - **`from typing import List, Optional`** 하고 **`def first(lst: List[str]) -> Optional[str]:`** 처럼 "문자열 리스트를 받아서, 첫 번째를 반환하거나 없으면 None" 같은 함수를 하나 만들어 보면 된다. Phase 1 Day 1부터 이런 식으로 시그니처를 쓰게 된다.
- **완료 조건**: 인자와 반환값에 타입을 붙인 함수를 최소 하나 만들고, 실행해 보았다.

---

## 📌 참고

- **typing**: **함수 시그니처에 타입을 적어 두는 것**이다. **`def f(s: str) -> int:`** 처럼 쓰고, **`typing`** 모듈에서 **`List`**, **`Dict`**, **`Optional`** 등을 가져와 **`List[int]`**, **`Optional[str]`** 처럼 쓴다. Phase 1부터 코드 템플릿과 예제에서 이렇게 쓰므로, "인자/반환 타입을 이렇게 표시한다"만 알아 두면 된다.
