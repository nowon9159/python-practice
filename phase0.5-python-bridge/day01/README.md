# Day 1: class 기초 — __init__, self, 속성, 메서드

## 🎯 오늘의 목표

- **클래스(class)**가 뭔지 알아 보고, **`__init__`**, **self**, **속성**, **메서드**를 한 번 써 본다.

---

## ✅ 오늘 할 일 (순서대로)

### 1. 클래스 하나 정의하기

- **할 일**: `day01/` 안에 `class_basics.py`를 만든다.
  - **`class Person:`** 을 정의한다.
  - **`def __init__(self, name, age):`** 에서 `self.name = name`, `self.age = age` 로 속성을 저장한다.
  - 메서드 **`def greet(self):`** 를 만들어 **`return f"Hello, {self.name}"`** 처럼 반환하게 한다.
  - 아래에서 **`p = Person("홍길동", 20)`** 으로 만들고 **`print(p.greet())`** 가 "Hello, 홍길동"이 나오는지 확인한다.
- **완료 조건**: `Person` 인스턴스를 하나 만들고, `greet()` 를 호출해 출력했다.

---

## 📌 참고

- **class**: **데이터(속성)와 동작(메서드)를 하나로 묶는 단위**다. 예를 들어 "사람"을 이름·나이(데이터)와 인사하기(동작)로 묶어 두면, 나중에 `Person("홍길동", 20)` 처럼 하나의 "객체"로 만들 수 있다.
- **__init__**: 인스턴스를 **만들 때** 한 번 호출된다. 여기서 **`self.속성명 = 값`** 으로 그 인스턴스만의 데이터를 둔다. `Person("홍길동", 20)` 을 하면 `__init__(self, "홍길동", 20)` 이 호출되고, `self.name`, `self.age` 가 저장된다.
- **self**: 메서드의 **첫 번째 인자**로, "지금 이 인스턴스"를 가리킨다. 속성에 접근할 때 **`self.name`**, **`self.age`** 처럼 쓴다.
