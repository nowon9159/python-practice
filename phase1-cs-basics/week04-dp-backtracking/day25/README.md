---

## ✅ 오늘 할 일 (순서대로)

### 1. 백트래킹 템플릿 정리

- **할 일**: `day25/` 안에 `backtrack_template.md`(또는 주석)를 만든다. **서브셋 / 조합 / 순열** 중 최소 2가지에 대해 다음을 적는다.
  - **상태**: 현재 인덱스(또는 시작 인덱스), 현재 path(리스트).
  - **base case**: 조건 만족 시 결과 리스트에 path 복사본 추가.
  - **for 루프**: 선택 → path에 추가 → 재귀 → path에서 제거(undo).
- **완료 조건**: "선택 → 재귀 → 복원"이 한 사이클로 정리되어 있다.

### 2. Subsets 풀이

- **할 일**: `phase1-cs-basics/algorithms/backtracking/`(또는 `day25/`)에 `subsets.py`를 만든다.
  - **문제**: 중복 없는 정수 배열의 모든 부분집합(서브셋) 반환. 백트래킹으로 각 인덱스에서 "넣음/안 넣음" 선택.
  - **조건**: docstring, 테스트 케이스 3개 이상.
- **완료 조건**: 로컬/제출 테스트 통과.

### 3. Combination Sum 풀이

- **할 일**: 같은 폴더에 `combination_sum.py`를 만든다.
  - **문제**: 같은 원소 여러 번 쓸 수 있을 때 합이 target이 되는 조합 전부. 백트래킹에서 "같은 인덱스 다시 선택 가능"하도록 인덱스 유지 또는 재귀 인자 설계.
  - **조건**: docstring, 테스트 케이스 3개 이상.
- **완료 조건**: 로컬/제출 테스트 통과.

### 4. N-Queens 풀이

- **할 일**: 같은 폴더에 `n_queens.py`를 만든다.
  - **문제**: n×n 체스판에 퀸 n개를 서로 공격하지 않게 놓는 모든 배치. 행마다 한 퀸, 열/대각선 충돌 체크.
  - **조건**: docstring, 테스트 케이스 n=1, 4 등.
- **완료 조건**: 로컬/제출 테스트 통과.
