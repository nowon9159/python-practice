## ✅ 오늘 할 일 (순서대로)

### 1. Big-O 분석 연습

- **할 일**: `day02/` 안에 `bigo_notes.md`(또는 파이썬 파일)를 만들고, **아래 중 최소 2개**에 대해 시간·공간 복잡도를 Big-O로 적는다.
  - 단일 루프 1개인 코드 한 조각
  - 중첩 루프 2개인 코드 한 조각
  - 재귀 함수 하나 (몇 번 호출되는지, 깊이 등 간단히)
- **완료 조건**: 각 코드에 대해 "O(?) 시간, O(?) 공간"과 이유 1줄이 적혀 있다.

### 2. Fibonacci 세 가지 버전 구현

- **할 일**: `day02/` 안에 `fibonacci.py`를 만들고 **세 가지 구현**을 넣는다.
  1. **재귀**: `fib(n)` = `fib(n-1) + fib(n-2)`, base case `n <= 1`.
  2. **메모이제이션**: 위 재귀 + 캐시(딕셔너리 등)로 중복 호출 제거.
  3. **Tabulation**: 반복문 + 리스트(또는 변수 2개)로 bottom-up.
- **완료 조건**: 세 함수가 모두 같은 `n`에 대해 같은 값을 반환하고, `if __name__ == "__main__"`에서 `n=10`, `n=30` 등으로 실행해 결과를 확인했다. (선택) 각 버전의 시간복잡도를 주석으로 적어 둔다.

### 3. Anki 카드 추가

- **할 일**: Big-O 표기법(최악/상수 무시/dominant term), 재귀(base case / recurrence), "재귀→DP 전환" 중 기억하고 싶은 항목을 Anki 카드로 **최소 2장** 만든다.
- **완료 조건**: Week 1 덱에 위 주제 카드가 추가되어 있다.

## 📌 참고 (복습할 개념)

- **Big-O**: 최악/평균/최선, 상수 무시, dominant term
- **재귀**: 종료 조건(base case), recurrence, 스택 프레임
- **재귀→DP**: 중복 계산 제거 → 메모이제이션 또는 Tabulation
