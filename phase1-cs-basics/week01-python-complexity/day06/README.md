## ✅ 오늘 할 일 (순서대로)

### 1. 패턴별 백지 재현

- **할 일**: **아래 5개 패턴** 각각에 대해, 코드 없이 **의사코드 또는 핵심 로직 3~5줄**을 백지(또는 `day06/review.md`)에 쓴다. 참고 없이 5분 안에 한 패턴씩 써 보기.
  1. 타입힌트·dataclass·collections 사용 예 (어떤 상황에 뭘 쓰는지)
  2. Big-O·재귀·Fibonacci 메모이제이션/Tabulation 흐름
  3. Two Pointers vs Sliding Window (언제 어떤 걸 쓰는지, 예시 1개)
  4. Hashing (Two Sum, Group Anagrams, Top K에서 키/값으로 뭘 썼는지)
  5. Stack·Monotonic Stack (Valid Parentheses, Daily Temperatures 핵심 동작)
- **완료 조건**: 5개 패턴 모두 한 번씩 백지로 적었고, 막힌 부분은 교재/노트로 채웠다.

### 2. Anki 카드 추가·수정

- **할 일**: Week 1에서 빠진 패턴이 있으면 **패턴명, 조건, 시간복잡도, DevOps 예시(선택)** 형식으로 카드를 추가한다. 기존 카드 중 헷갈리는 것은 뒷면을 보강한다.
- **완료 조건**: Week 1 덱에 Day 1~5 패턴이 골고루 반영되어 있다.

### 3. 막힌 문제 재풀이

- **할 일**: Day 3~5 중 **한 번에 풀지 못했던 문제**를 1~2개 골라, 힌트/해설 최소한만 보고 다시 풀어 본다. 풀이는 기존 파일에 덧붙이거나 새 파일로 저장.
- **완료 조건**: 선택한 문제를 스스로 다시 풀었고, 왜 막혔었는지 1줄 메모해 둔다.
