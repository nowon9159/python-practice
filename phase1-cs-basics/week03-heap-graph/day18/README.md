## ✅ 오늘 할 일 (순서대로)

### 1. Kahn's Algorithm 구현

- **할 일**: `day18/` 안에 `topological_sort.py`를 만든다.
  - **입력**: DAG를 인접 리스트로 (노드 → 나가는 이웃 리스트). 노드 개수 V.
  - **동작**: (1) 모든 노드의 **in-degree** 계산. (2) in-degree 0인 노드를 **큐**에 넣음. (3) 큐에서 하나씩 꺼내서 결과에 추가, 그 노드에서 나가는 간선의 끝점 in-degree 1 감소, 0이 되면 큐에 넣음. (4) 결과 길이가 V가 아니면 **사이클 존재** → 위상정렬 불가(에러 또는 빈 리스트 반환 규칙 정하기).
  - **출력**: 위상정렬된 노드 리스트(또는 불가 시 빈 리스트/예외).
- **완료 조건**: DAG 예시 2개에 대해 올바른 순서가 나오고, 사이클 있는 그래프는 "불가"로 처리된다.

### 2. CI/CD 파이프라인 의존성 정렬기

- **할 일**: 같은 폴더에 `cicd_dependency_order.py`를 만든다.
  - **입력**: Job 이름 목록과 "Job A는 Job B 이후에 실행" 같은 의존 관계 리스트 (예: `[("deploy", "test"), ("test", "build")]`).
  - **동작**: 위 관계를 그래프로 만들고(의존되는 쪽 → 의존하는 쪽 간선), **Kahn's Algorithm**으로 실행 가능한 순서를 출력. 사이클이 있으면 "의존성 순환 있음" 등 메시지 출력.
  - **실행 예**: `uv run python cicd_dependency_order.py` 또는 인자/파일로 의존 관계 입력.
- **완료 조건**: 의존 관계를 넣으면 올바른 실행 순서가 출력되고, 순환 의존성이 있으면 감지된다.
