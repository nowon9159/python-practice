# Day 71: Python 심화 — 메타클래스, 디스크립터, __slots__

## 🎯 오늘의 목표

메타클래스(클래스의 클래스, type, __new__), 디스크립터(__get__, __set__), __slots__를 이해하고 각각 예제 코드를 작성해 차이를 설명할 수 있게 한다.

### 1. 메타클래스 예제

- **할 일**: `phase3-advanced/week09-10-algo/day71/` 안에 `metaclass_example.py`를 만든다. **메타클래스**를 사용한 예제 하나를 구현한다. (예: 클래스 생성 시 자동 등록하는 레지스트리 패턴 — `type` 또는 `__new__`로 클래스 생성 시 리스트/딕셔너리에 등록.)

- **할 일**: 같은 폴더에 `descriptor_example.py`를 만든다. **디스크립터** 클래스( `__get__`, `__set__` 구현)를 하나 만들고, 이를 사용하는 클래스의 속성으로 쓴다. (예: 타입 검사/validator가 있는 attribute, 또는 ORM 스타일 필드.)

## 📌 참고 (복습할 개념)

- 메타클래스: 클래스의 클래스, type, __new__로 클래스 생성 시 커스터마이즈
- 디스크립터: __get__/__set__, property/validator, ORM 필드
- __slots__: 인스턴스 __dict__ 제거, 고정 속성만 허용, 메모리 절약 (Day 35 연결)
