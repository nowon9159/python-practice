# Day 85: 전체 커리큘럼 회고 — 패턴 카탈로그 최종 정리

## 🎯 오늘의 목표

90일 전체 패턴·개념을 한 번에 정리한 **패턴 카탈로그** 문서를 최종화하고, Phase 1~3 요약과 "문제 유형 → 패턴" 매핑을 완성한다.

---

## ✅ 오늘 할 일 (순서대로)

### 1. 패턴 카탈로그 문서 작성

- **할 일**: `docs/pattern-catalog-final.md`(또는 `phase3-advanced/week11-12-final/day85/pattern-catalog-final.md`)를 만든다. **각 패턴**에 대해 다음을 적는다: **조건**(언제 쓰는가), **시간복잡도**, **대표 문제 1개**, **DevOps 연결 1줄**. Phase 1(배열/해시/스택/큐/트리/그래프/DP/백트래킹), Phase 2(프로세스/소켓/분산/DB/캐시/설계), Phase 3(고급 알고리즘, Python 심화, 설계 심화)를 모두 포함한다.
- **완료 조건**: 문서에 Phase 1~3 패턴이 정리되어 있고, 각 패턴당 조건·복잡도·대표 문제·DevOps 1줄이 있다.

### 2. 무작위 10개 패턴 백지 설명 테스트

- **할 일**: 패턴 카탈로그에서 **무작위 10개**를 뽑아, **백지**에서 각각 1분씩 말로 설명해 본다. (녹음 또는 메모.) 막힌 패턴은 카탈로그에서 다시 읽고 표시해 둔다.
- **완료 조건**: 10개 패턴에 대해 백지 설명을 시도했고, 막힌 패턴이 표시되어 있다.

### 3. Anki 덱 구조·Day 86 준비

- **할 일**: Anki 덱 구조를 한 번 점검한다. (덱 이름, 카드 수, 반복할 카드.) Day 86 Feynman 최종 테스트(무작위 개념 20개)용으로 카탈로그/Anki에서 20개를 뽑을 수 있게 목록을 준비해 둔다.
- **완료 조건**: 패턴 카탈로그 문서가 완성되었고, 무작위 10개 백지 테스트를 했으며, Day 86에서 쓸 20개 개념 풀 목록이 준비되어 있다.
