# Day 81: 알고리즘 취약 패턴 집중 보강

## 🎯 오늘의 목표

지금까지 취약했던 패턴 2~3개를 선정하고, 같은 유형 문제 2~3개씩 풀이하며 패턴별 "핵심 코드" 한 블록을 백지에서 재현할 수 있게 한다.

---

## ✅ 오늘 할 일 (순서대로)

### 1. 취약 패턴 2~3개 선정

- **할 일**: Day 67 갭 분석·모의 인터뷰 복기에서 나온 **취약 패턴 2~3개**를 선정한다. `phase3-advanced/week11-12-final/day81/` 안에 `weak_patterns.md`에 패턴명·선정 이유(몇 번 막혔는지)를 적는다.
- **완료 조건**: 문서에 취약 패턴 2~3개가 명시되어 있다.

### 2. 패턴당 2~3문제 풀이

- **할 일**: NeetCode/자기 노트 기반으로 **각 패턴당 2~3문제**를 풀이(또는 복습)한다. 풀이는 `day81/`에 패턴별로 저장한다. (예: `sliding_window_1.py`, `sliding_window_2.py`.) 각 풀이에 **패턴별 "핵심 코드" 한 블록**을 주석으로 표시한다.
- **완료 조건**: 선정한 2~3개 패턴 각각에 대해 2~3문제 풀이가 있고, 핵심 코드가 주석으로 표시되어 있다.

### 3. 핵심 코드 백지 재현·Anki

- **할 일**: 패턴별로 **핵심 코드 한 블록**을 백지에서 다시 써 본다. (다른 파일에 `template_패턴명.py`로 저장해도 됨.) Anki 해당 패턴 카드를 복습한다.
- **완료 조건**: 취약 패턴 문제 풀이가 완료되었고, 해당 Anki 카드를 복습했으며, Day 82 Blind 75 복습 준비가 되어 있다.
