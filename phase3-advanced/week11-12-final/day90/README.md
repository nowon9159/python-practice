# Day 90: Phase 3 완료 회고 + 다음 90일 계획

## 🎯 오늘의 목표

90일 커리큘럼 완료 회고 문서를 작성하고, 다음 90일(또는 단기) 학습 계획과 패턴 카탈로그·Anki 유지보수 방침을 정리한다.

---

## ✅ 오늘 할 일 (순서대로)

### 1. 90일 회고 문서 작성

- **할 일**: `docs/retro-90days.md`(또는 `phase3-advanced/week11-12-final/day90/retro-90days.md`)를 만든다. **잘한 점**, **아쉬운 점**, **유지할 습관**을 적는다. **Phase 1~3 중 가장 도움 된 부분**을 1~2문단으로 쓴다.
- **완료 조건**: 회고 문서에 위 항목이 포함되어 있고, 90일을 돌아볼 수 있는 분량이다.

### 2. 다음 90일(또는 다음 단계) 목표 정리

- **할 일**: 같은 문서 또는 `next_90days.md`에 **다음 90일(또는 단기) 목표**를 적는다. 예: NeetCode 150 완주, 오픈소스 PR 1개, 시스템 설계 주 1회 연습 등. 구체적으로 3~5개 목표를 나열한다.
- **완료 조건**: 다음 계획 목록이 문서에 있고, 실행 가능한 수준으로 적혀 있다.

### 3. 패턴 카탈로그·Anki 유지보수 방침

- **할 일**: **패턴 카탈로그**와 **Anki**를 어떻게 유지할지 방침을 적는다. (예: 주 1회 Anki 리뷰, 월 1회 패턴 카탈로그 보강 등.) 회고 문서 마지막 섹션 또는 별도 1줄로 저장한다.
- **완료 조건**: 90일 회고 문서가 완성되었고, 다음 계획 목록과 유지보수 방침이 정리되었으며, 90일 완료를 확인했다.
