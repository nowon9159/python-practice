# Day 82: NeetCode Blind 75 빠른 복습 (타이머 세션)

## 🎯 오늘의 목표

Blind 75(또는 NeetCode 150 핵심) 리스트를 한 번 훑으며 문제당 5~10분 안에 접근법만 떠올리거나 코드 스케치를 하고, 패턴 인식 속도를 높인다.

---

## ✅ 오늘 할 일 (순서대로)

### 1. Blind 75(또는 선정 목록) 순회

- **할 일**: **Blind 75** 또는 사용 중인 핵심 문제 리스트를 순회한다. 각 문제에 **5~10분**만 할당한다. "이건 어떤 패턴?" → 의사코드 또는 핵심 식만 적기. 전부 풀기보다 **패턴 인식 속도**를 올리는 것이 목표. 막히면 바로 해설 확인 후 다음 문제. `phase3-advanced/week11-12-final/day82/` 안에 `blinds75_notes.md`를 만들고 **각 문제별 패턴명·핵심 아이디어 1줄**을 메모한다.
- **완료 조건**: 목록 1회전이 완료되었고, 대부분 문제에 패턴명·핵심 아이디어 1줄이 메모되어 있다.

### 2. 취약 문제 목록 업데이트

- **할 일**: 완전히 잊었거나 패턴을 못 떠올린 문제를 **취약 문제 목록**에 표시한다. (blinds75_notes.md에 ✅/❌ 또는 "재풀이 필요" 표시.) 나중에 재풀이할 목록을 5~10개 정도 정리한다.
- **완료 조건**: 취약 문제 목록이 업데이트되어 있고, Day 83 최종 프로젝트 전에 참고할 수 있다.

### 3. Day 83 최종 프로젝트 준비

- **할 일**: 커리큘럼에서 Day 83 **분산 태스크 큐** 미니 프로젝트 내용을 읽고, 사용할 폴더(`phase3-advanced/final-project/` 등)와 구현할 함수(태스크 정의, enqueue, worker 루프)를 한 번 정리한다.
- **완료 조건**: Blind 75 1회전·취약 목록 정리가 끝났고, Day 83에서 쓸 폴더·구조를 파악했다.
