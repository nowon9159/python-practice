# Day 89: 전체 Anki 덱 리뷰

## 🎯 오늘의 목표

90일 동안 만든 Anki 카드 전체(또는 핵심 카드)를 1회 리뷰하고, 까먹은 카드는 복습하며 불필요한 카드는 정리한다.

---

## ✅ 오늘 할 일 (순서대로)

### 1. Anki 덱 리뷰 세션

- **할 일**: **전체 Anki 덱**을 열고 가능한 범위까지 **1회 리뷰**한다. "앞면만 보고 뒷면이 바로 떠오르는가?"를 기준으로 표시한다. 반복해서 막히는 카드는 **요약 수정** 또는 **카드 분할**을 한다.
- **완료 조건**: 덱 1회전이 완료되었거나, 핵심 카드만이라도 1회 리뷰했다.

### 2. 카드 정리

- **할 일**: **유지할 카드**와 **수정·삭제할 카드**를 정리한다. 수정한 카드는 저장하고, 불필요한 카드는 삭제 또는 보관 처리한다. `phase3-advanced/week11-12-final/day89/` 안에 `anki_review_notes.md`에 "수정한 카드 주제", "삭제/보관한 카드 수" 등을 한 줄씩 적어 둘 수 있다.
- **완료 조건**: 카드 정리가 반영되었고, (선택) 리뷰 노트가 저장되어 있다.

### 3. Day 90 회고 초안

- **할 일**: Day 90에서 쓸 **90일 요약 초안**을 한두 줄이라도 생각해 둔다. (잘한 점, 아쉬운 점, 유지할 습관, Phase 1~3 중 가장 도움 된 부분 중 1~2개.)
- **완료 조건**: Anki 덱 1회전(또는 핵심만)과 카드 정리가 끝났고, 90일 회고 시 참고할 초안 메모가 있다.
